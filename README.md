#VCF-TO-CLINVAR
A web-based application for determining clinically significant variants from a human VCF file.

## Justification
No web-based tool exists for finding ClinVar entries associated with variants of clinical significance from a VCF file. Existing tools for this task require a command line interface, which some researchers, inclduing students, may not be familiar with. This tool addresses that gap and provides a simple, easy to use tool for finding variants of clinical relevance in a VCF file.
