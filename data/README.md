# Data Directory

This directory contains all data used by the VCF-to-ClinVar annotation pipeline.
No data files are committed to the repository. All files are generated locally
by running `just build-all` from the project root.

---

## Directory Structure

```
data/
├── raw/          # Original downloaded files — never modified
├── demo/         # Streamlit app demo inputs — final pipeline outputs
├── processed/    # Intermediate transformed files
└── test/         # Hand-curated fixtures for unit testing
```

## Data Sources

| Field       | Details |

## File Descriptions

### `data/raw/`

| File | Description |
|------|-------------|
| `variant_summary.txt.gz` | ClinVar archive — kept after decompression |
| `variant_summary.txt` | ClinVar full variant summary TSV — source for all annotation |
| `ALL.chr21.GRCh38.phased.vcf.gz` | 1000 Genomes chr21 archive — kept after decompression |
| `ALL.chr21.GRCh38.phased.vcf` | Full chr21 VCF — source for subsampling |

### `data/processed/`

| File | Description | Generated by |
|------|-------------|--------------|
| `chr21_subsampled.vcf` | Every 50th variant from the full chr21 VCF (~20,000 variants) | `just subsample-vcf` |
| `chr21_with_pathogenic.vcf` | Subsampled VCF with known Pathogenic variants appended | `just append-pathogenic` |

### `data/demo/`

| File | Description | Generated by |
|------|-------------|--------------|
| `clinvar_chr21.tsv` | ClinVar filtered to GRCh38 chr21 only — used for all merges | `just filter-clinvar-chr21` |
| `demo.vcf` | Subsampled VCF filtered to ClinVar-matched variants only | `just filter-clinvar` |

### `data/test/`

| File | Description | Source |
|------|-------------|--------|
| `test_real.vcf` | Minimal hand-crafted VCF for unit testing the parser | Manual |
| `known_pathogenic_chr21.vcf` | 10 confirmed Pathogenic chr21 variants from ClinVar | Manually curated from ClinVar |

## Reproducing the Data

New teammates can generate all data from scratch with a single command:

```bash
just build-all
```

This runs the following pipeline in order:

```
setup → install → download → filter-clinvar-chr21 → subsample-vcf → filter-clinvar
```

Individual steps can also be run in isolation — see `just --list` for all available commands.

---

## Notes

- `data/raw/` files are large (several GB) and are excluded from git via `.gitignore`
- `data/processed/` and `data/demo/` files are also excluded — regenerate with `just build-all`
- `data/test/` fixtures are small and **committed to the repository**
- ClinVar is filtered to GRCh38 assembly and chromosome 21 only for performance
- The 1000 Genomes VCF contains healthy population samples — Pathogenic variant matches are expected to be rare
